📊 Sales Data ETL Pipeline & Power BI Dashboard
🚀 Project Overview

This project demonstrates an end-to-end Sales Analytics solution by building an automated ETL pipeline using Python. It extracts raw sales data, performs cleaning and transformation, loads it into a SQLite database, and visualizes key business insights using Power BI dashboards.

The goal is to convert raw data into meaningful insights for better business decision-making.

⚙️ Technologies Used
Python (Pandas)
SQL (SQLite)
Power BI
DAX
Windows Task Scheduler
🔄 Project Workflow
1. Extract
Load raw sales data from CSV files using Python
2. Transform
Handle missing and invalid data
Clean and structure datasets
Calculate key business metrics like Total Sales
3. Load
Store processed data into SQLite database tables
4. Automate
Schedule ETL pipeline using Windows Task Scheduler for automation
5. Visualize
Build interactive Power BI dashboards
Create KPIs, charts, and business insights
📈 Key Features
Automated ETL pipeline from raw data to insights
Data cleaning and preprocessing using Pandas
KPI generation for business analysis
SQLite database integration
Interactive Power BI dashboard for reporting
Fully automated execution using scheduler
📊 Dashboard Insights
Total Sales Performance
Product-wise Sales Analysis
Customer Trends
Time-based Sales Patterns
Business KPIs overview
🎯 Project Objective

To simulate a real-world data analytics pipeline that transforms raw business data into actionable insights using modern BI tools and automation.
